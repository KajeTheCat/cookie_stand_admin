# cookie_stand_admin

small app to quickly show practice into tailwindcss and react. stringified JSON and displayed after entering data into the four fields and clicking the create button. Added hover state to button to change background black and text white. Added custom hex coded colors to config file for color display on form, header, footer, and button.
